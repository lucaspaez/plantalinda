# Cannabis App Frontend

This is a [Next.js](https://nextjs.org/) project bootstrapped manually.

## Prerequisites

*   **Node.js**: You must have Node.js installed to run this application. [Download here](https://nodejs.org/).

## Getting Started

1.  Open a terminal in this directory (`frontend`).
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

*   `src/app`: App Router pages and layouts.
*   `src/components`: Reusable UI components.
*   `public`: Static assets.
